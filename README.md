# Survival1XP

Minecraft Paper/Bukkit плагин для хардкорного выживания с 1 HP.

## Особенности
- Игрок имеет только 1 HP (полсердца)
- HUD скорборд с прогрессом и заданиями
- 7 заданий для прохождения игры
- Автоматический спектатор режим при смерти
- Поддержка русского и английского языков (по локали клиента)

## Команды
- `/1xp` - список всех команд
- `/stats` - вкл/выкл HUD
- `/dead` - вернуться в игру из спектатора
- `/rs` - сбросить свою статистику
- `/quest` - описание всех заданий
- `/reset` - полный сброс сервера (OP)

## Задания
1. Деревня - найти деревню и поторговать с жителем
2. Незер - зайти в Незер
3. Стержни - получить огненный стержень
4. Жемчуг - получить жемчуг эндера
5. Око эндера - скрафтить око эндера
6. Посетил Энд - зайти в Энд
7. Убил дракона - победить Эндер Дракона

## Установка
1. Скачать JAR из Releases
2. Положить в папку `plugins` сервера
3. Перезапустить сервер

## Сборка
```bash
mvn clean package
```

## Требования
- Paper/Spigot 1.21+
- Java 21+
